elbanwop
====================
BIN400 challenge released at Hack In The Box Amsterdam 2017 CTF.
Implements a custom VM interpreter in pure ROP.

Dependencies
====================

* gcc
* python
* [ropstone](https://github.com/blasty/ropstone)

Building
====================

```
$ make
```


Running
====================

```
$ ./vuln input.bin
```
